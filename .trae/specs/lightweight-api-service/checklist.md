# 轻量级API服务 - 验证清单

## 功能验证

### 项目与环境管理
- [ ] 能够创建、编辑、删除项目
- [ ] 能够创建、编辑、删除API分组
- [ ] 能够创建、编辑、删除环境变量
- [ ] 能够在不同环境间切换

### 第三方API管理
- [ ] 能够录入第三方API的基本信息（方法、URL、名称、描述）
- [ ] 能够配置第三方API的请求参数（Headers、Query Params、Path Variables、Body）
- [ ] 能够配置第三方API的鉴权信息（Bearer Token、API Key、Basic Auth）
- [ ] 能够引用环境变量作为鉴权信息

### 本地API配置
- [ ] 能够创建本地API，设置自定义路径和请求方法
- [ ] 能够选择处理模式（Mock Mode或Proxy Mode）
- [ ] 能够配置静态Mock数据
- [ ] 能够绑定第三方API进行转发

### Mock功能
- [ ] 静态Mock模式下，能够返回预设的JSON数据
- [ ] 动态Mock模式下，能够生成随机数据
- [ ] 条件Mock模式下，能够根据入参返回不同数据

### 转发功能
- [ ] 透明转发模式下，能够将请求原封不动转发给第三方API
- [ ] 转换模式下，能够将本地入参映射为第三方API的字段
- [ ] 能够注入参数或Header
- [ ] 能够处理第三方API的响应并返回给调用方
- [ ] 防SSRF攻击机制有效

### 在线调试面板
- [ ] 能够选择本地API进行测试
- [ ] 能够填写参数并发送请求
- [ ] 能够友好展示响应结果（Response Body、Headers、HTTP Status Code、耗时）
- [ ] 能够生成cURL、Fetch、Axios、Python等代码片段

### 日志与监控
- [ ] 能够记录每一次对本地API的请求日志
- [ ] 能够查看转发详情（原始请求 -> 转换后请求 -> 第三方请求 -> 第三方响应 -> 转换后响应）
- [ ] 能够查询和筛选日志

## 非功能验证

- [ ] 系统能够支持高并发处理
